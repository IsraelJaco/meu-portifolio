Sistema de Gestão de Estoque

Gerencie seus produtos com praticidade e eficiência

Descrição

Este sistema de gestão de estoque foi desenvolvido com foco em simplicidade, desempenho e praticidade. Ideal para pequenas e médias empresas, permite o controle completo de produtos, entradas e saídas, e emissão de relatórios.

Funcionalidades

  • ✅ Cadastro de produtos com categorias e códigos
  • 📦 Registro de entrada e saída de estoque
  • 📊 Relatórios dinâmicos e exportação
  • 🔐 Login e controle de usuários
  • 💾 Banco de dados local com backup

Tecnologias Utilizadas

  • 🖥️ Interface em Java Swing (NetBeans)
  • 💾 Banco de dados SQLite
  • 🧠 Arquitetura MVC
